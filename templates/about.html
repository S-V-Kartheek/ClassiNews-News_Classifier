About ClassiNews

Understanding how our AI-powered news classifier works

What is ClassiNews?

ClassiNews is a news classification tool that automatically sorts articles into categories using machine learning.

Just paste in a headline or full article and it'll tell you if it's World News, Sports, Business, or Sci/Tech.

How It Works

1

Text Preprocessing

We clean up the text - lowercase it, remove URLs, and strip out stopwords.

2

Feature Extraction

TF-IDF converts the text into numbers the model can work with.

3

Classification

The model looks at the features and picks the best category.

Technology Stack

Backend: Python, Flask
ML Framework: scikit-learn
NLP: NLTK, TF-IDF Vectorization
Model: Multinomial Naive Bayes
Frontend: HTML5, CSS3, JavaScript
Deployment: Flask Web Server

Privacy & Security

We store predictions locally in a JSON file. No personal info is collected.

Everything runs on the server - keeping it simple with no databases or external services.
